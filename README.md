# 비트코인 삼성전자 주식 web app deployed on Heroku


The deployed web app is live at https://btcsamsungapp.herokuapp.com/

streamlit을 알게되어서 테스트용 저장소입니다.

+ streamlit==0.61.0
+ pandas==0.25.3
+ numpy==1.18.1
+ cryptocmd==0.6.0
+ pandas-datareader==0.9.0
+ plotly==4.14.3
+ DateTime==4.3
